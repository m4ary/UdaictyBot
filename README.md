## :zap: deploy your own

### Requirements
- Telegram Bot Token ([create yours](https://core.telegram.org/bots#3-how-do-i-create-a-bot))
- JVM 1.8

### How to:
1. replace`YOUR_BOT_TOKEN`in`Consts`Java Class with your token bot token.
